My Projects Some Of My Works

Now Platform

This "Now Platform" project involved creating a user-friendly e-commerce website using React. The website was designed to be visually appealing and easy to navigate. Advanced React concepts such as state management, class-based components, and Context API were utilized to build the website's functionality. The website also incorporated user-friendly modules for categories, products, and store management, along with a picker feature for efficient order processing.

Name It

The "Nameit" project is a React-based application that generates over 100 name suggestions tailored to user-specified keywords. It allows users to seamlessly check the availability of corresponding domain names on namecheap.com with a simple click. This project demonstrates proficiency in React, API integration, and user experience design, providing a valuable tool for brainstorming and validating domain name ideas.

Sonic Speed

Sonic Speed Typing is a React-based game that tests your typing skills. You'll race against the clock to type random text prompts as quickly and accurately as possible. The game tracks your typing speed in words per minute, counts your characters, and highlights incorrect letters in red. It's a fun way to improve your typing speed and challenge yourself.

Monster Killer Game

The "Monster Player" project is a text-based game built with HTML, CSS, and JavaScript. Players combat various monsters, managing their own health and the monster's. Featuring basic attack options like attack, strong attack, and healing Powers, the project allows for exploration of game mechanics and UI/UX design principles with HTML elements and styling. This project demonstrates proficiency in core web development concepts and explores the fundamentals of game development logic.
